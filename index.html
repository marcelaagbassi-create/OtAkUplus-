❓ Aide OtAkU+

🎬 Regarder un anime

Clique sur une affiche → choisis la saison → clique sur un épisode. Utilise le sélecteur de langue VOSTFR / VF.

📚 Lire un manga

Va dans Bibliothèque → clique sur un manga → choisis un chapitre → LIRE.

❤️ Favoris

Clique sur le cœur 🤍 sur une carte pour ajouter aux favoris.

📺 Mode TV

Utilise les flèches de la télécommande. Le curseur orange suit tes mouvements. Appuie sur OK/Enter pour sélectionner.

🌙 Veille auto

Après 3 min d'inactivité, diaporama automatique. Appuie sur une touche pour revenir.
